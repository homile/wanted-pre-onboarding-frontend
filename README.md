## git branch & commit

- branch

  - main: 메인 배포
  - dev: 개발
  - feature/auth: 로그인/회원가입 기능 및 디자인 구현
  - feature/todo: Todo List 기능 및 디자인 구현

- commit
  - feat: 기능 구현
  - fix: 기능 수정
  - style: 디자인 구현
  - chore: 파일 삭제 및 코드 정리
  - docs: 문서 수정
  - refactore: 코드리팩토링
